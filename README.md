## 按需求选套餐：一个直接的建议

不同用途对线路的要求差很多，硬套"最便宜"这个标准很容易买错。

**面向海外用户（目标受众在欧美或全球）**：LAX T1 系列完全够用，国际 Tier 1 路由覆盖广。年付 WEE 最省钱，月付 TINY $6.9 也不贵。

**国内用户访问速度有要求，但预算有限**：优先蹲 LAX Pro WEE 的补货（$36.9/年，CN2 GIA 线路），或者选 LAX EB TINY（$9.99/月），CMIN2 + 9929 对联通移动用户效果不差。

**对中国访问延迟要求高（电信用户为主，或做面向国内用户的业务）**：LAX Pro 系列是更稳妥的选择。价格贵，但 CN2 GIA 双向优化是这家做得最扎实的地方，晚高峰基本不掉速。实测延迟在 140-180ms 之间，相比同价位竞品有优势。

**需要日本节点或面向日本/东南亚用户建站**：TYO T1 入门年付 $36.9 起，亚太路由优化，延迟表现对周边用户友好。TYO Pro 的 CN2 GIA 则适合同时需要对接中国用户的场景。

**高防需求（建站经常遭受 DDoS 攻击）**：圣何塞 SJC T1 系列自带 20Gbps DDoS 防护，价格在 T1 系列里也算合理，这个防护能力基本不需要额外付费。

## 购买前需要知道的几件事

**1. 默认密钥登录，不支持密码登录。** 购买后需要通过 SSH 密钥文件连接服务器。如果不熟悉这个流程，官网有教程，按步骤来不难，但第一次容易卡。

**2. 退款政策。** 3 天内且流量使用不超过 30GB，可全额退款（扣支付手续费）；30 天内可按比例退。不确定的话先买入门款测试，退款有保障，风险不高。

**3. IP 被墙可以换。** 每 15 天可以免费申请换一次 IP（被墙情况下）。非被墙原因换 IP 收费 $5。对做梯子的用户来说，这个政策比那种换 IP 要催好几天才能处理的商家强。

**4. 套餐升级可以，降级麻烦。** 买了之后想升配置可以联系客服处理，但降级要重新购买。第一次入手建议稍低一档先跑跑看。

**5. 热门套餐经常缺货。** DMIT 不超售，所以配置有多少就卖多少，Pro WEE 和 EB WEE 这种性价比特别突出的套餐经常一放就没。如果看上了不在库里，官网可以看到"Notify Me"入口，留邮件等补货通知。

## 常见问题（FAQ）

**Q：DMIT 最便宜的套餐是哪个？**

A：LAX T1 WEE，年付 $36.9，折合每月不到 $3.1。没有中国大陆优化，适合面向海外用户的场景。如果需要中国优化，LAX Pro WEE 同价，但经常缺货；EB WEE 偶尔以 $39.9/年的价格出现。

**Q：T1 系列国内访问到底能不能用？**

A：可以用，但不稳定。白天还好，晚高峰（9-11 点）容易有延迟抖动，丢包率比优化线路高。如果只是偶尔需要国内访问、主要业务面向海外，T1 够用。对国内访问质量有要求的就别在 T1 上委屈自己了。

**Q：香港 T1 和洛杉矶 T1 同价，选哪个？**

A：看你的用户在哪里。香港 T1 走国际线路，没有 CN2 GIA，但物理上更靠近亚太，对东南亚用户延迟更低。面向中国大陆用户的话，香港 T1 不如香港 Pro；面向全球用户，LAX T1 在北美覆盖更好。两者同价，用途决定选哪个。

**Q：Pro 系列真的比 EB 好很多吗？**

A：Pro 是三网 CN2 GIA 双向优化，EB 是 CMIN2 + 9929 回程。在电信用户场景下 Pro 明显好；联通移动用户两者差距没那么大。EB 价格比 Pro 低一档，对预算有限的联通移动用户来说是更合适的入门选项。

**Q：退款靠谱吗？**

A：3 天内全额（扣手续费）、30 天内比例退款这个政策是写在官方条款里的。用过的人反馈实际退款流程没遇到太多阻力。不确定要不要买，先入一个最便宜的套餐测试，真不合适退了就是。
